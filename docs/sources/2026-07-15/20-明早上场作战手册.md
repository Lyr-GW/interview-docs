## 6 · 应急：被问到没写过的题

> **完整 12 场景话术库**（不会 / 没测过 / 数字对不上 / 代码抓包 / 竞品没写过 / 问穿了 / 要客户日志 / 追 LOC / 追加速比 …）：见 [`39-高压追问应急话术卡`](./39-高压追问应急话术卡.md)。  
> 下表只留三步框架 + 3 个短例；上场高压优先翻 `39`。

**框架三步**（边界诚实 → 迁移已知 → 反问场景）：

1. **边界诚实（5–10s）**  
   > 「这个具体题/论文我没在生产里落地过，我不瞎编实现细节。」

2. **迁移已知（30–45s）**  
   从本夜/简历主场抽 **同构问题**：  
   - 调度/缓存类 → Motor precise vs 近似树（`12`/`17`）或 Dynamo cost（`01` A）  
   - 约束/正确性 → xgrammar bitmask + async 错位（`16`/`19`）  
   - 传输/PD → 选型 `24` + handoff `07` + Mooncake `13` + HCCL 边界 `36`；ZMQ 通知面 `25`  
   - 性能归因 → Profiling 四层（`01` D）；TTFT 五段/测算 → **`38`**  
   讲清：**问题形态、取舍轴、失败模式**，不假装读过对方仓库。

3. **反问场景（10s）**  
   > 「你们更关心延迟、吞吐，还是正确性/多租户？我按那个目标对齐一版设计权衡。」

**短例**（更多见 `39`）

| 被问 | 诚实 | 迁移 | 反问 |
|------|------|------|------|
| 某新投机论文细节 | 没跟到实现 | draft-verify + 接受率掉点（`01` B） | 低并发加速还是大 batch 稳定？ |
| 某 KV 池化产品名 | 没部署过 | Conductor 真索引 vs 本地树（`12`/`13`） | 要精确命中还是近似省中心？ |
| 某算子融合名 | 没写过该 kernel | Roofline + FA/CG 边界（`06`） | 算力墙还是带宽墙？ |
| 客户日志 / −70% | 无 raw log | 测算白板 **`38`** | 复测要分位还是机制？ |

**禁止**：用「应该差不多」编 API；用假 LOC/假加速比填沉默。
