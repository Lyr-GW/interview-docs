## 3 · 出门前禁语（7:55 · 必做）

对照 `20` §5 / [`69`](./69-禁语20条默背卡.md) / `03` / `58`，出声勾：

- [ ] FIFO/100，不是 LRU/128  
- [ ] −70% = **测算**，不是客户 raw log  
- [ ] MTP × SO = **互斥**  
- [ ] D **不注册** Conductor  
- [ ] bitmask = **torch 组合**，非自研 fused  
- [ ] guidance = **预留**；线上仅 xgrammar  
- [ ] LOC / 加速比：**待补 / 未压测**  
- [ ] 跳槽：**拉力>推力**，非涨薪嫌加班  
- [ ] Connector ≠ Mooncake 官方三层（`69` #11）  
- [ ] 200ms = **配置超时**，不是已测 P99（`69` #14）  

有余量再扫 `69` 全文 20 条；时间紧只勾上表。

**30 秒口令**：

```text
100 FIFO → 测算非日志 → torch 组合 → MTP 互斥 → D 不注册
→ 不吹 kernel → 见十五 → 拉力大于推力
```
