## 1 · 题面（盖住文末再答）

### R · 简历绑定（5）

**R1.** 假命中：近似树「字符前缀很长」为什么仍可能灌到冷副本？Motor 精确路径靠什么事件感知驱逐？Stored 与 Removed 各解决什么？

**R2.** Herding：三版演进（overlay → top-k → unified）各修什么病？`load_gated` 和 `unified` 选型一句；默认五参数你能背出哪几个？

**R3.** PD：何时选混部 U，何时 P+D？handoff 与 concurrent/layerwise 的 TTFT 直觉差在哪？**D 实例为什么不注册 Conductor？**

**R4.** 异步调度 × 约束解码：mask/采样错位的三类根因？修法验收标准为什么是「正确性」而不是「TPOT −x%」？

**R5.** NPU bitmask：你们是 torch 组合路径还是 vLLM 那种 fused？能报自研 kernel / μs 级实测吗？MTP 与结构化为什么入口硬互斥？

### J · JD 盲区（5）

**J1.** Chunked Prefill：要解决的 HOL 是什么？和抢占/recompute 怎么分工？和 PD 分离的收益边界一句怎么划？

**J2.** SGLang RadixTree：`match_prefix` / 节点分裂 / `lock_ref` 各一句；树命中能否等同「投机接受」？和 Spec V2 overlap 别搅在哪？

**J3.** 语义路由 / 强弱模型：实例级 vs 模型级怎么框？难度信号 δ 怎么来？与 Motor KV 亲和是正交还是互相替代？

**J4.** MoE：EP / All2All / MC2 各一句；「128 选 6、8K」每专家均值 token 数白板怎么算？能报自家 MoE 加速比吗？

**J5.** Decode 为何想开 CUDA Graph / aclgraph？与 PagedAttention（变长 `block_table`）的矛盾怎么解？K8s 三探针先后 + RAS「单一 owner」你怎么一句带过？

### W · 白板（3）

**W1.** 画 MindIE schema 编译缓存：**100 + FIFO + 命中不调序**；标出「新插入从哪进、踢谁、命中指针动不动」。

**W2.** 默写 Motor `unified`（或你最熟的）打分式：前缀分 / 负载分怎么进；再写 top-k 防 herding 时「只在候选集内重排」的意思。

**W3.** 二选一默写：**(a)** TTFT≈\(c_0+T_{prefill}(1-h)\) 并标 h 从 0.10→0.78 的故事；或 **(b)** TP Column vs Row 各一次集合通信（AllGather / AllReduce）直觉图。

### S · 软性（2）

**S1.** 「你的缺点是什么？」请 technic 化：选一个**已有补强路径**的弱项（如投机曾失守 / 配置曾零输出），60s 讲「暴露 → 怎么补 → 今晚卡片号」，禁止空洞「太追求完美」。

**S2.** 「和同事/跨团队意见冲突怎么处理？」给一个**推理栈交付**场景（接口边界 / 口径 / oncall）的 60s STAR；结尾落到可验证证据，不落到情绪。
