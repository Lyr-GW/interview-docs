    推理算子学习索引与覆盖清单¶

    本目录是「没写过算子的推理框架开发者」补齐算子层盲区的学习包。 目标：覆盖推理优化岗位对算子/硬件/图模式的常见追问；诚实边界：框架主战场 + Roofline/读源码撑追问，不谎称独立交付 AscendC/HCCL。

    2. Decode Step 算子地图（把零散知识钉在时间线上）¶

    Embed → RMSNorm → QKV proj → RoPE → KV write → Attention → O proj
      → RMSNorm → FFN/MoE → … ×L 层 → LM Head → Sampler
    Step 阶段 算子层在干什么 本仓主落点 常见瓶颈
    0 Embed 查表 22 通常小
    1 Pre-Attn Norm Vector 归一化 02；常融入 qkv_rms_norm_rope_cache 单独做则 Host/小算子开销大
    2 QKV proj Cube MatMul/GEMV 01；MLA：mla_preprocess/prolog；TP：mc2/* Decode 访存密集
    3 RoPE Vector 旋转 02；apply_rotary_pos_emb / 融入 1–2 小算子，宜融合
    4 KV write 写 Paged/连续 cache 03；scatter_pa_kv_cache、rope_quant_kvcache 带宽
    5 Attention FA 分块 + online softmax 03；Prefill:PFA Decode:IFA 统一:FIA Prefill 算力 / Decode 带宽
    6 O proj MatMul + 常 AllReduce 01；matmul_all_reduce（04 MC2） TP 通信
    7 Post Norm Vector 02；常融入 *_add_rms_norm 同 1
    8a Dense FFN GateUp+SwiGLU+Down 01；ffn/ffn、grouped_matmul_swiglu_quant Decode 访存；大 batch 可翻计算
    8b MoE gating→permute→GMM→finalize 04 EP 通信 + 专家负载
    9 LM Head MatMul（N=V） 18 大 V 带宽；vocab parallel
    10 Sampler (+mask) Vector / 图外 15、11、23 Host/错位/约束
