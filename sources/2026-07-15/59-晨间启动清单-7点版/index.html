    2 · 抽卷 5 题怎么抽（链 56/57/68）¶

    1. 打开 56（或 57 / 68），盖住文末卡号表。
    2. 简历 / JD 盲区 / 白板 / 软性 尽量各沾一点（不必严格 5 区满配）。
    3. 答完再翻卡号；错题只回炉对应短卡（03/39/32/62…），不读专题全文。
    4. 昨夜已刷过 A → 改抽 B 或 C 5 题；勿 A+B+C 连刷满 45。
    5. 想专刷后半夜刀口（传/Store/超时/五参数）→ 优先 68。

    3 · 出门前禁语（7:55 · 必做）¶

    对照 20 §5 / 69 / 03 / 58，出声勾：

    • [ ] FIFO/100，不是 LRU/128
    • [ ] −70% = 测算，不是客户 raw log
    • [ ] MTP × SO = 互斥
    • [ ] D 不注册 Conductor
    • [ ] bitmask = torch 组合，非自研 fused
    • [ ] guidance = 预留；线上仅 xgrammar
    • [ ] LOC / 加速比：待补 / 未压测
    • [ ] 跳槽：拉力>推力，非涨薪嫌加班
    • [ ] Connector ≠ Mooncake 官方三层（69 #11）
    • [ ] 200ms = 配置超时，不是已测 P99（69 #14）

    有余量再扫 69 全文 20 条；时间紧只勾上表。

    30 秒口令：

    100 FIFO → 测算非日志 → torch 组合 → MTP 互斥 → D 不注册
→ 不吹 kernel → 见十五 → 拉力大于推力

    被抓包改口 → 39 总纲，不临场编。
