    56 · 随机抽题卷 A（抽签默背）¶

    本夜续批（2026-07-15 · 04:22）
    用途：模拟面试官「抽签」——只给题面，不给答案；答完再翻文末卡号对答案。
    构成：简历绑定 5 + JD 盲区 5 + 白板 3 + 软性 2 = 15 题。
    用法：骰子/闭眼点号 → 出声 45–90s（白板可写）→ 盖住文末 再对卡。
    姊妹卷：57 · 68；收官路径：49。
    口径冲突以 03 为准；数字性质先 50。

    0 · 抽签规则（30 秒）¶

    1. 随机抽 5–8 题即可（全刷约 25–35′）；A/B 卷勿同夜连刷两套满题。
    2. 卡壳超过 20s → 记题号，先跳；全部答完再翻「参考翻卡号」。
    3. 禁止开卷偷看 01/32 正文；翻卡仅用于对完后纠偏。
    4. 口述踩红线（LRU/128、客户实测 −70%、吹 kernel）→ 当场用 03 八条自罚重答。
    区 题号 目标
    R 简历绑定 R1–R5 从简历句挖第三层
    J JD 盲区 J1–J5 平安失分/JD 常考、简历未写深
    W 白板 W1–W3 默写公式/四步/五段
    S 软性 S1–S2 动机/高压救场

    建议抽法（任选）：闭眼点题号；或 shuf -e R1 R2 R3 R4 R5 J1 J2 J3 J4 J5 W1 W2 W3 S1 S2 | head -6。
    计时：R/J/S 题 ≤90s；W 题写完再讲 ≤3′。超时记「半对」，翻卡后重答一遍。

    1 · 题面（盖住文末再答）¶

    R · 简历绑定（5）¶

    R1. 简历写 schema 编译缓存。默认容量多少？淘汰策略是 FIFO 还是 LRU？命中会不会调序？为什么不是「热 schema 更抗踢」？

    R2. 「TTFT −70%」是客户线上 raw log 实测，还是代表性测算？请用一句话钉死性质，并说出公式里主砍的是哪一段（不要说成 FlashAttention 魔法）。

    R3. 结构化输出约束解码：为什么说是字节级 PDA 而不是随口 FSM？guidance 在你们仓里是「真多后端」还是「接口预留」？

    R4. Motor KV 亲和：tokenize 为什么前置到 Coordinator？tools 注入时字符级树 vs token 级索引可能怎样错位？fail-closed 指什么？

    R5. Server C++ 重构 /「约 1 万行」/ SO「5000+」：现状 wc 能否直接当累计证据？被追 LOC 时你的诚实边界是哪三词？

    J · JD 盲区（5）¶

    J1. 投机解码：拒绝采样为什么说「无损」？大 batch 下草稿接受率掉了，你会怎么一句话解释，而不是报假加速比？

    J2. 面试官点名「上周新论文 / DSpark」：它和 MTP、DFlash 各差在哪一句？MindIE 三插件你能背出哪三个名字？

    J3. vLLM 加速配置（二面零分区）：按目标（降 TTFT / 提吞吐 / 省显存）各举两类旋钮，并各用半句讲「背后原理」，不要只报旗名。

    J4. Mooncake：Conductor / Store / Transfer Engine 各干什么？Motor 在这条链上只做哪一步决策？Connector 是不是官方三层之一？

    J5. Dynamo：cost 公式白板里有哪几项直觉？KVBM G1–G4 你怎么和 Motor「查索引再打分」一句对标（猜 vs 查）？

    W · 白板（3）¶

    W1. 默写拒绝采样四步（draft → verify → accept/reject → …）。标出「哪一步保证与原模型同分布」。

    W2. 默写 TTFT 五段分解；在旁标注 −70% 测算主要砍哪一段、哪些假设必须说出口。

    W3. 默写 Dynamo（或等价）routing cost 直觉式：latency / load / KV hit 怎么进式子；再写 Motor unified 打分里你记得住的权重项。

    S · 软性（2）¶

    S1. 「为什么离开华为 / 为什么来我们这？」请用 拉力 > 推力 结构答 60s；禁止涨薪、嫌加班作为主句。

    S2. 高压追问：「你简历数字是不是包装的？」给 2–3 句应急模板（测算 / 待补 / 未压测），不要道歉式崩溃。
